# Install EA

This repository stores the MT5 Expert Advisor source at:

```text
mt5/Experts/MTChartBridge/MTChartBridgeEA.mq5
```

To install it manually:

1. Copy `MTChartBridgeEA.mq5` into the MT5 `MQL5/Experts/MTChartBridge/` folder.
2. Open MetaEditor.
3. Compile the `.mq5` file locally.
4. Attach the EA to a chart in MetaTrader 5.
5. Configure the local shared folder input according to the EA settings.

Generated `.ex5` files are local build outputs and must not be committed.
