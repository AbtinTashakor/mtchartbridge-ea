# MTChartBridge EA

MTChartBridge EA is an open-source MetaTrader 5 Expert Advisor core for a local chart-to-terminal bridge.

It is not a trading robot. It does not generate trading signals, connect to the internet, use DLLs, use sockets, or call WebRequest.

The intended architecture is:

```text
Chrome Extension <-> Local Shared Folder <-> MT5 Expert Advisor
```

The Chrome Extension writes command files into a user-selected local folder. The MT5 Expert Advisor reads those local files, validates commands, calculates risk-based volume inside MT5, and writes response files back to the same local workflow.

Current status: **Phase 5 Risk Engine / Volume Calculation / No Trade**.

Phase 5 keeps Phase 4 market validation and then calculates account-equity-based risk, stop-loss loss per lot with `OrderCalcProfit`, and a broker-step-normalized volume. It can reject commands whose requested risk is too high, too small for the broker minimum volume, or impossible to calculate safely. It still does not execute trades.

## Safety Boundaries

- No trading signals are generated by this repository.
- No final trade volume is accepted from the extension.
- Final trade volume is calculated inside MT5 from `risk_percent`, account equity, current Bid/Ask reference price, stop loss, and broker volume constraints.
- Phase 5 performs validation and risk calculation only; it does not import `CTrade`, call `OrderSend`, or open positions.
- No generated `.ex5` files are committed.
- No MT5 runtime folders or Common/Files runtime data are committed.
