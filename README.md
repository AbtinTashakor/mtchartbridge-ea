# MTChartBridge EA

MTChartBridge EA is an open-source MetaTrader 5 Expert Advisor core for a local chart-to-terminal bridge.

It is not a trading robot. It does not generate trading signals, connect to the internet, use DLLs, use sockets, or call WebRequest.

The intended architecture is:

```text
Chrome Extension <-> Local Shared Folder <-> MT5 Expert Advisor
```

The Chrome Extension writes command files into a user-selected local folder. The MT5 Expert Advisor reads those local files, validates commands, and writes response files back to the same local workflow. Final trade volume is the EA's responsibility and will be calculated inside MT5 in a later phase.

Current status: **Phase 4 Market Validation / No Risk / No Trade**.

Phase 4 validates `trade.open` commands against live MT5 symbol and market state after protocol validation passes. It can reject unavailable symbols, disabled trading permissions, unavailable prices, high spread, invalid SL/TP side, and broker stop-level violations. It still does not execute trades and does not calculate final risk-based volume yet.

## Safety Boundaries

- No trading signals are generated by this repository.
- No final trade volume is accepted from the extension.
- Final trade volume will be calculated inside MT5 in a later phase.
- Phase 4 performs market validation only; it does not import `CTrade`, call `OrderSend`, or open positions.
- No generated `.ex5` files are committed.
- No MT5 runtime folders or Common/Files runtime data are committed.
