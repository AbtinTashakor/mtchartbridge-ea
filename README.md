# MTChartBridge EA

MTChartBridge EA is an open-source MetaTrader 5 Expert Advisor core for a local chart-to-terminal bridge.

It is not a trading robot. It does not generate trading signals, connect to the internet, use DLLs, use sockets, or call WebRequest.

The intended architecture is:

```text
Chrome Extension <-> Local Shared Folder <-> MT5 Expert Advisor
```

The Chrome Extension writes command files into a user-selected local folder. The MT5 Expert Advisor reads those local files, validates commands, calculates risk-based volume inside MT5, runs a no-trade execution safety check, and writes response files back to the same local workflow.

Current status: **Phase 6 Execution Safety Check / OrderCheck / No Trade**.

Phase 6 keeps Phase 5 MT5-side risk calculation, requires `dry_run=true`, builds an `MqlTradeRequest` preview for a market order, and optionally runs `OrderCheck`. It never calls `OrderSend`, never imports `CTrade`, and never creates an order, deal, ticket, or position.

## Safety Boundaries

- No trading signals are generated by this repository.
- No final trade volume is accepted from the extension.
- Final trade volume is calculated inside MT5 from `risk_percent`, account equity, current Bid/Ask reference price, stop loss, and broker volume constraints.
- Phase 6 performs validation, risk calculation, and no-trade execution checking only; it does not import `CTrade`, call `OrderSend`, or open positions.
- No generated `.ex5` files are committed.
- No MT5 runtime folders or Common/Files runtime data are committed.
